## SIA-GAN
#### Signal and Image Attention GAN

## 개요
#### ECG Signal과 STFT를 통해 이미지화 된 ECG를 이용한 GAN 기반의 multimodal 심부전 탐지 알고리즘

## 목표
#### signal과 freq의 feature를 transformer를 활용해 효과적으로 fusion하는 GAN기반의 심부전 탐지 알고리즘 개발

## 파일 구조
### experiments
 - 1학기동안 진행한 다양한 실험들

### final_model
 - 실험 후에 얻은 SIA-GAN의 최종 모델
 - AUC : 0.9588
